Lets look at the example. In HTML, we create rows and add data cells to them. By adding more data cells in each row, we create more columns.


A table with one <th> row and three <td> rows.

What if the table contains data that spans multiple rows and columns? Data can span rows and columns using the rowspan and colspan attributes. The attribute accepts an integer (greater than or equal to 1) to denote the number of rows or columns it spans across. Here is another example:

<td colspan='2'>Adam's Greenworks</td>
